Personal Projects

Super ASGE Bros

I am currently in the process of recreating the Nintendo Classic, Super Mario Bros from the NES in C++.
Built in the ASGE engine, an OpenGL based project with bare bones functionallity, I display an understanding of Component based Arcatecture as well as Factory and State Machine patterns. This project is the one I'm currently working on and very proud of, and is the best example of my current C++ programming skills.

Pixel Moon

A personal unity project where I'm working with JRPG battle mechanics and trying to recreate the visual art style of the Square Enix title, Octopath Traveler. The project currently has a working battle system that generates an initiative for each member at the start of each round. It then uses this and sorts a list of states to get the turn order.
