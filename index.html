Personal Projects

Super ASGE Bros - View on github

In this project I am currently in the process of recreating the Nintendo classic, Super Mario Bros from the NES.
This is being built in ASGE, an OpenGL based game engine with bare bones functionallity, supplied by my University. In this project I display an understanding of Component based Arcatecture as well as Factory and State Machine patterns. This project is the one I'm currently working on and very proud of, and is the best example of my current C++ programming skills.

Pixel Moon

A personal unity project where I'm working with JRPG battle mechanics and trying to recreate the visual art style of the Square enix title, Octopath Traveler.
